# Mobile To-Do App

A mobile-first to-do list application built with HTML, CSS, and JavaScript.

## Live Demo
https://pasindumendis-tech.github.io/mobile-todo-app/

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Mobile-friendly UI

## Technologies Used
- HTML5
- CSS3
- JavaScript
- GitHub Pages

## Purpose
This project demonstrates frontend state management, event handling, and mobile-first UI design.
